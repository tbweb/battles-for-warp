This class handles the ships currently played, with other importants infos,
such as the player who own the ships, their positions on the map,
the id of the game, etc.

An instance must be declared with three parameters : the name of the ship,
his position x and his position y.

This class provides the following methods:

Int			getId();
Return the _id value.

PlayShip	setId($var);
Set the _id variable to the parameter given and return the current instance.

Int			getIdGame();
Return the _id_game value.

PlayShip	setIdGame($var);
Set the _id_game variable to the parameter given and return the current instance.

Int			getIdPlayer();
Return the _id_player value.

PlayShip	setIdPlayer($var);
Set the _id_player variable to the parameter given and return the current instance.

Int			getIdShip();
Return the _id_ship value.

PlayShip	setIdShip($var);
Set the _id_ship variable to the parameter given and return the current instance.

Ship		getShip();
Return the _ship value.

PlayShip	setShip($var);
Set the _ship variable to the parameter given and return the current instance.


String		getShipName();
Return the _ship_name value.

PlayShip	setShipName($var);
Set the _ship_name variable to the parameter given and return the current instance.

String		getWay();
Return the _way value.

PlayShip	setWay($var);
Set the _way variable to the parameter given and return the current instance.

Int		getX();
Return the _x value.

PlayShip	setX($var);
Set the _x variable to the parameter given and return the current instance.

Int		getY();
Return the _y value.

PlayShip	setY($var);
Set the _y variable to the parameter given and return the current instance.

Int		getPc();
Return the _pc value.

PlayShip	setPc($var);
Set the _pc variable to the parameter given and return the current instance.

Int		getShield();
Return the _shield value.

PlayShip	setShield($var);
Set the _shield variable to the parameter given and return the current instance.

Int		getInertia();
Return the _inertia value.

PlayShip	setInertia($var);
Set the _inertia variable to the parameter given and return the current instance.

Int		getPlayed();
Return the _played value.

PlayShip	setPlayed($var);
Set the _played variable to the parameter given and return the current instance.

A static variable $verbose can be triggered to TRUE to diplsay some debug infos.
